Basic JavaScript programming 3

You can define functions in JavaScript. A function is a section of code which can be called from another part of your code. Functions are useful for encapsulating code that you want to call repeatedly without copying and pasting it everywhere. In the sample script.js there are two functions defined but neither are called when the page is loaded. Instead, we are using HTML to call the functions when a button on the page is pressed.

Also in this script you can see quite a few new concepts - we are using the "DOM" to find a particular thing on our page (in this case, a drawable object called a canvas). Objects in JavaScript encapsulate behaviour - you can ask an object about itself or ask it to do something. In our case, we are asking it to clear or draw squares (really, rectangles, but squares are the easy case).
