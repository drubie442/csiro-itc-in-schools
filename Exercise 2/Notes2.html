Basic JavaScript programming 2

Programming languages use different techniques to keep information and control the flow of the program itself. Information inside a program can be broadly classed into static (or constant) information and variables. There is a third class of information we won"t deal with here which is data that comes from an external source like a database, or an XML file, or a network source.

Constants

Constants can be built in to a language, or you can define them yourself. JavaScript doesn't have a standard definition for constants (i.e. information used in the program that you don't want to change). In other programming languages it is possible to define something with a keyword (typically "const") which means that if you try to change the value to something else, your program will generate an error.

Variables

Variables are values you expect will change during program execution. You use variables as counters, as text output (called a "string") or other specialised numbers (integers, double precision numbers, boolean values). In our sample program we have two different variables (message and counter). In this program, message is not modified, just output. The counter variable is used to keep track of our while loop.

Control flow

The program, when running, starts at the top of the script.js file and continues downwards until it hits the "while" loop. A while loop is a smaller piece of code which executes multiple times until some condition is achieved. In this case, the condition is that the counter must always be less than the value of 10. If the counter ever reaches 10, the while loop will not execute and the program will skip over the section inside the curly braces and continue down the page until there is no more code to run. If we didn't set an achievable condition on the while loop, it will run until somebody stops the program externally or the computer is switched off.

There are quite a few different types of control flow in most programming languages although they are all of a few different useful types:

If-then-else conditions

Used when you want to test for some condition and then conditionally do something once or not at all (or you can cascade an if statement). if (counter == 10) { ... } else { ... }

while loops

There is an example while loop in the script.js file

for loops

A for loop is a piece of code you wish to execute exactly N many times. If you know ahead of time that your program should do something 5 times in a row (for example), you would use a for loop: for (var i=0; i < 5; i++) { ... }

switch statements

Usually used when you have some condition that has a discrete number of states and you need to do something different for each state.switch (myvariable) { case "booyah": ... break; case "boohoo": ... break; }

Keywords

The conditional, control flow and variable declaration words (if, var, while etc) are all examples of keywords. All programming languages have a set number of keywords which have special meaning for that programming language. In JavaScript the terminology is "reserved words"

JavaScript reserved words

You can"t use reserved words as names for variables (it confuses the language parser).
